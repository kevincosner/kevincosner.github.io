Homework Assignment #2: Shading [PDF]

Major Task

  • Implement vertex and fragment shaders as well as a CPU program to perform Phong shading with the Phong lighting model data described in a Material Template File (*.mtl) and render the model on the screen.

Grading Policy

  • Loading the material data correctly: 35% (download test files)
  • Implement Phong shading with a point light correctly: 15%
  • Implement Phong shading with a directional light correctly: 15%
  • Implement Phong shading with a spot light correctly: 15%
  • Code organization: 10%
  • Report: 5%
  • Bonus: 5% (control lights with keyboard or UI)
